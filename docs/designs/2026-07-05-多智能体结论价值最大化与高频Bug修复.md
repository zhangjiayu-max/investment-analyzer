### 1.2 5 个高价值增强点

**Bug A — percentile 字符串致任务失败**（rebalancing + market_intelligence）
- 根因：DB 中 12 条历史 percentile 是文本型（"97.25%"），虽 `list_valuation_indexes()` 在读取时做 normalize，但部分消费点未受保护
- 影响：每天触发 rebalancing/market_intel 任务都失败，浪费 token + 用户得不到分析

**Bug B — cross_review 仅产出 tool_call 文本**（多智能体场景）
- 根因：[multi_agent.py](file:///Users/xiaoyuer/projects/investment-analyzer/backend/agent/multi_agent.py#L478-L484) 清理逻辑：`if cleaned and len(cleaned) > len(answer) * 0.3: answer = cleaned`，当 LLM 返回纯 tool_call 文本时 cleaned 为空，条件不满足，answer 保持原始 tool_call 文本
- 影响：cross_review 标记 success 但无分析内容，下游仲裁拿不到有效输入

**Bug C — ai_dialogue 路径不持久化结论**
- 根因：[db/analysis_conclusions.py#L105](file:///Users/xiaoyuer/projects/investment-analyzer/backend/db/analysis_conclusions.py#L105) `save_analysis_conclusion` 仅被 5 个独立分析路由调用（diversification/deep_dive/daily_report/bond_recommend/panorama），orchestrator/multi_agent 从未调用
- 影响：用户花 50 万 token/天做多智能体对话，结论只活在内存中，下一次对话又重新分析

**Bug D — cross_system_references 表空置**
- 根因：建表但无任何 insert 调用
- 影响：跨系统决策无法追溯（如 dashboard 的 decision_records 看不到对应的 agent 分析依据）

**Bug E — 长对话无超时保护**
- 根因：orchestrate() 主循环没有总时长限制，单个 chat_turn 可达 8 分钟
- 影响：用户体验差、token 失控

### P0-1：修复 percentile 文本型致任务失败（Bug A）

**策略：写入防御 + 消费防御 + 数据迁移** 三层兜底

#### 1. 数据迁移（一次性，启动时执行）

在 [db/_conn.py](file:///Users/xiaoyuer/projects/investment-analyzer/backend/db/_conn.py) 的 `init_db()` 末尾追加迁移逻辑（幂等）：

```python
def _migrate_text_percentile_to_real():
    """把 index_valuations.percentile 历史文本型数据转为 REAL（幂等）。"""
    conn = _get_conn()
    rows = conn.execute(
        "SELECT id, percentile FROM index_valuations WHERE typeof(percentile)='text'"
    ).fetchall()
    if not rows:
        return  # 已迁移
    for r in rows:
        raw = r["percentile"]
        try:
            val = float(str(raw).replace('%', '').strip())
        except (ValueError, TypeError):
            val = None
        conn.execute(
            "UPDATE index_valuations SET percentile=? WHERE id=?",
            (val, r["id"])
        )
    conn.commit()
    conn.close()
    logger.info(f"[migration] 已把 {len(rows)} 条文本型 percentile 转为 REAL")
```

```python
# L99 附近
def _safe_float(v, default=None):
    """percentile 防御性 float 转换。"""
    if v is None or v == "":
        return default
    if isinstance(v, (int, float)):
        return float(v)
    try:
        return float(str(v).replace('%', '').strip())
    except (ValueError, TypeError):
        return default

# L99: percentile = v.get("percentile")  →
percentile = _safe_float(v.get("percentile"))
# L218: return val.get("percentile", 50) or 50  →
return _safe_float(val.get("percentile"), 50) or 50
```

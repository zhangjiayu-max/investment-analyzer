> 参考有知有行、韭圈儿、蚂蚁财富等优秀产品，设计值得吸收的功能增强。

## 一、对标分析

### 1.1 有知有行 — "四笔钱"资产配置框架

**核心理念：** 按资金用途分桶，而非按产品分类。

| 桶 | 用途 | 期限 | 风险偏好 | 产品举例 |
|---|---|---|---|---|
| 活钱管理 | 日常开支 | 随时 | 极低 | 货币基金 |
| 稳健理财 | 中短期目标 | 1-3年 | 低 | 债券基金、固收+ |
| 长期投资 | 财富增值 | 3年+ | 中高 | 指数基金、主动基金 |
| 保险保障 | 风险兜底 | - | - | 重疾、医疗、寿险 |

**市场温度计：** 根据估值百分位动态调整权益/固收比例。低估时增加权益（如80%），高估时降低（如40%）。

**可吸收点：**
- 持仓自动归类到四笔钱，展示每笔钱的占比和健康度
- 市场温度计 → 我们已有估值数据，缺的是动态配比建议

### 1.2 韭圈儿 — 专业工具集

**恐贪指数：** 0-100的情绪指标，基于换手率、涨停家数、融资余额等计算。5以下极度恐慌，95以上极度贪婪。

**股债性价比（FED模型）：** 沪深300 PE倒数 vs 10年国债收益率。差值越大，股票越有吸引力。

**组合回测：** 任意组合的历史收益、最大回撤、夏普比率、年化波动率。

**基金账本：** 记录每笔买入/卖出，计算真实收益率（考虑分红、赎回费）。

**可吸收点：**
- 恐贪指数 → 我们已有市场情绪数据，可以做一个类似的0-100指数
- 股债性价比 → 新增，数据源已有
- 滚动收益分析 → 核心功能，下面详细设计

### 1.3 蚂蚁财富 — "理财分"（321模型）

**"321"评估框架：**
- **3角理念：** 选品 + 配置 + 持有
- **2轮演算：** 定性评估 + 数据智能校准
- **1个分数：** 综合行为健康度（满分1000，600+为健康）

**评分维度：**
| 维度 | 指标 | 权重 |
|---|---|---|
| 选品 | 是否持有优质基金（同类排名前50%） | 高 |
| 配置 | 股债分散度、行业分散度 | 高 |
| 持有 | 持有时长、交易频率 | 高 |
| 定投 | 是否有定投习惯 | 中 |
| 仓位 | 单只基金仓位是否过重 | 中 |

**关键数据：**
- 600分以上用户收益显著优于600以下
- 高风险用户：未持优质基金 vs 持有多只优质基金，收益率差23%
- 仅持单只基金 vs 股债配置，波动率高45%
- 高频交易 vs 中频交易，收益率低4%

**可吸收点：**
- 综合理财健康分 → 核心功能，下面详细设计

## 二、新增功能设计

### P0：综合理财健康分（核心）

**目标：** 一个分数概括用户的投资行为健康度，类似蚂蚁财富的"理财分"。

**评分框架（满分1000分）：**

| 维度 | 权重 | 评分标准 | 数据来源 |
|---|---|---|---|
| **选品质量** | 200分 | 持有基金的同类排名分位、基金经理评分 | akshare + 估值数据 |
| **分散配置** | 200分 | 股债比例合理性、行业分散度、相关性矩阵 | 相关性分析已有 |
| **估值合理性** | 200分 | 持仓整体估值百分位（低估加权） | 估值数据已有 |
| **持有行为** | 200分 | 平均持有时长、交易频率、追涨杀跌次数 | 交易记录 |
| **风控纪律** | 200分 | 最大回撤控制、止损执行、仓位管理 | 持仓快照 |

**输出格式：**
```
┌─────────────────────────────────────┐
│       你的投资健康分：720/1000       │
│           ▓▓▓▓▓▓▓░░░ 72%            │
├─────────────────────────────────────┤
│  选品质量  ████████░░  168/200      │
│  分散配置  ██████░░░░  120/200      │
│  估值合理  ████████░░  172/200      │
│  持有行为  ███████░░░  140/200      │
│  风控纪律  ██████░░░░  120/200      │
├─────────────────────────────────────┤
│  📋 改进建议：                       │
│  1. 增加债券基金配置，降低相关性     │
│  2. 当前交易频率偏高（月均4.2次）    │
│  3. 单只基金仓位22%过重，建议<15%   │
└─────────────────────────────────────┘
```

### P1：滚动收益分析

**目标：** 回答"任意时点买入，持有N年能赚多少？"

**核心指标：**
- 任意时点买入持有1年/3年/5年的收益分布
- 胜率（正收益概率）
- 最大收益、最小收益、中位收益
- 年化收益中位数

**输出：**
```
中证白酒指数 滚动收益分析（2015-2026）

持有1年：胜率 62% | 中位收益 +8.3% | 最差 -35%
持有3年：胜率 78% | 中位收益 +28.6% | 最差 -12%
持有5年：胜率 91% | 中位收益 +62.4% | 最差 +5%

💡 结论：持有3年以上，正收益概率显著提升
```

### P1：股债性价比（FED模型）

**目标：** 判断当前是否应该增配股票或债券。

**公式：**
```
股债性价比 = 沪深300 PE倒数（盈利收益率） - 10年国债收益率
```

**判断标准：**
- 差值 > 3%：股票极具吸引力，建议增配权益
- 差值 1-3%：股票有一定吸引力
- 差值 < 1%：股票吸引力低，建议增配债券
- 差值 < 0%：股票可能高估，谨慎

**数据源：** akshare 已有沪深300 PE 和国债收益率数据。

### P1：恐贪指数

**目标：** 0-100的情绪指标，辅助判断市场极端情绪。

**计算因子：**
| 因子 | 权重 | 数据源 |
|---|---|---|
| 沪深300近20日涨跌幅 | 20% | akshare |
| 涨跌家数比 | 20% | akshare |
| 换手率（vs历史均值） | 15% | akshare |
| 北向资金净流入 | 15% | akshare |
| 新基金发行热度 | 10% | akshare |
| 融资余额变化 | 10% | akshare |
| 搜索指数（百度/微信） | 10% | 可选 |

**输出：** 0-100指数 + 历史曲线 + 当前区间（极度恐慌/恐慌/中性/贪婪/极度贪婪）

### P2：四笔钱归类

**目标：** 自动将持仓归类到四笔钱框架。

**实现：**
- 货币基金 → 活钱管理
- 纯债基金 → 稳健理财
- 混合/股票/指数基金 → 长期投资
- 保险类 → 保险保障（手动标记）

**展示：**
- 每笔钱的总金额和占比
- 每笔钱的健康度（是否符合该桶的风险偏好）
- 配比建议（如"活钱偏多，可适当转入长期投资"）

### P2：定投优化器

**目标：** 根据估值和情绪，动态调整定投金额。

**策略：**
- 低估 + 恐慌：定投金额 × 1.5
- 正常：定投金额 × 1.0
- 高估 + 贪婪：定投金额 × 0.5 或暂停

**输出：** 每月定投计划表，包含每只基金的建议金额。

### 滚动收益计算

```python
def rolling_return(nav_series, holding_days):
    """计算滚动持有收益"""
    returns = []
    for i in range(len(nav_series) - holding_days):
        start = nav_series.iloc[i]
        end = nav_series.iloc[i + holding_days]
        ret = (end - start) / start
        returns.append(ret)
    return {
        'win_rate': sum(1 for r in returns if r > 0) / len(returns),
        'median': sorted(returns)[len(returns)//2],
        'max': max(returns),
        'min': min(returns),
        'count': len(returns),
    }
```

### 股债性价比

```python
def stock_bond_ratio():
    """FED 模型：股票盈利收益率 vs 国债收益率"""
    hs300_pe = get_index_valuation('沪深300')['pe']
    bond_yield = get_bond_yield_10y()  # 10年国债收益率
    earnings_yield = 1 / hs300_pe
    spread = earnings_yield - bond_yield / 100
    return {
        'earnings_yield': round(earnings_yield * 100, 2),
        'bond_yield': bond_yield,
        'spread': round(spread * 100, 2),
        'signal': '增配股票' if spread > 0.03 else '中性' if spread > 0.01 else '增配债券',
    }
```

## 核心算法（services/attribution.py）

```python
def brinson_attribution(
    holdings: list[dict],      # 期初持仓
    transactions: list[dict],  # 期间交易
    benchmark_weights: dict,   # 基准权重（按品类）
    start_date: str,
    end_date: str,
) -> dict:
    """Brinson 三因素分解
    
    Returns:
        {
            "total_return": float,           # 总收益
            "selection_effect": float,        # 选股效应
            "timing_effect": float,           # 择时效应
            "interaction_effect": float,      # 交互效应
            "category_attribution": [         # 品类归因
                {"category": "债券型", "return": 0.05, "contribution": 0.03, "weight": 0.6},
                ...
            ],
            "top_contributors": [...],        # 贡献最大的持仓
            "top_drags": [...],               # 拖累最大的持仓
        }
    """
```

### 算法步骤
1. 计算各品类期初权重 `wp` 和基准权重 `wb`
2. 计算各品类收益率 `Rp`（基于快照）和基准收益率 `Rb`
3. 选股效应 = Σ (wp × (Rp - Rb))
4. 择时效应 = Σ ((wp - wb) × Rb)
5. 交互效应 = Σ ((wp - wb) × (Rp - Rb))
6. 品类归因：按 fund_category 分组计算贡献度

def diagnose_behavior(user_id: str = "default", period_days: int = 90) -> dict:
    """行为偏差量化诊断
    
    Returns:
        {
            "disposition_effect": {           # 处置效应
                "score": 0.65,                # 0-1，越高越严重
                "sell_winners_ratio": 0.7,    # 卖盈占比
                "hold_losers_ratio": 0.8,     # 持亏占比
                "pgr": 1.2,                   # 盈利实现比率 PGR
                "plr": 0.4,                   # 亏损实现比率 PLR
                "detail": "倾向卖出盈利持仓，持有亏损持仓..."
            },
            "anchoring_effect": {             # 锚定效应
                "score": 0.45,
                "cost_reference_ratio": 0.8,  # 决策中引用成本价的比例
                "avg_hold_since_loss": 45,    # 亏损后平均持有天数
            },
            "herding_effect": {               # 羊群效应
                "score": 0.3,
                "hot_topic_buy_ratio": 0.4,   # 买入热点基金占比
                "avg_buy_after_peak_days": 5, # 高点后买入平均延迟
            },
            "overtrading": {                  # 过度交易
                "score": 0.55,
                "turnover_rate": 3.2,         # 年化换手率
                "avg_hold_days": 18,          # 平均持有天数
                "cost_ratio": 0.015,          # 交易成本占比
            },
            "overall_score": 0.49,            # 综合偏差分
            "suggestions": ["..."],           # 改善建议
        }
    """
```

### 算法步骤
1. **处置效应**：PGR = 卖出盈利数/(卖出盈利数+持有盈利数)，PLR 同理，DE = PGR - PLR
2. **锚定效应**：分析交易记录中"补仓/加仓"决策是否在亏损后发生，统计亏损后持有天数
3. **羊群效应**：对比买入日期与基金净值高点，计算"高点后买入"比例
4. **过度交易**：年化换手率 = Σ 卖出金额 / 平均持仓市值

def mean_reversion_analysis(index_code: str, metric_type: str = "市盈率") -> dict:
    """均值回归分析
    
    Returns:
        {
            "current_percentile": 0.35,
            "historical_mean": 15.2,
            "current_value": 12.3,
            "deviation_sigma": -1.2,        # 偏离均值的标准差
            "half_life_days": 120,           # 半衰期（回归均值速度）
            "reversion_probability": 0.75,   # 回归概率
            "expected_return_1y": 0.15,      # 预期1年收益
            "signal": "low_estimate",        # low_estimate|fair|high_estimate
            "signal_strength": 0.7,          # 0-1
        }
    """

def extreme_warning(index_code: str) -> dict:
    """极值预警
    
    Returns:
        {
            "is_extreme": True,
            "direction": "low",              # low|high
            "percentile": 0.05,
            "historical_frequency": 0.03,    # 历史上低于此值的比例
            "last_similar_date": "2018-12-28",
            "subsequent_return_1y": 0.25,    # 当时之后1年收益
            "suggestion": "历史极低位置，可考虑分批布局"
        }
    """

def forecast_signals() -> list[dict]:
    """全市场估值信号扫描（用于每日推送）"""
```

### 算法步骤
1. 取过去 5 年 PE 序列，计算均值 μ 和标准差 σ
2. 当前 PE 偏离度 = (current - μ) / σ
3. 半衰期：用 ORNstein-Uhlenbeck 过程拟合回归速度
4. 极值预警：percentile < 5% 或 > 95% 时触发
5. 历史类比：找历史上相似分位点的后续收益

- `attribution.enabled`（收益归因，默认 false）
- `behavior_diagnosis.enabled`（行为诊断，默认 false）
- `decision_accuracy.enabled`（决策准确率，默认 false）
- `strategy_backtest.enabled`（策略回测，默认 false）
- `portfolio_optimizer.enabled`（组合优化，默认 false）
- `valuation_forecast.enabled`（估值预测，默认 false）

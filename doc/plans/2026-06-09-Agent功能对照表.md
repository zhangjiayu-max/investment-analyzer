## 四、统一分析执行器

所有独立 Agent 分析可通过 `run_agent_analysis()` 统一调用：

```python
from agent.analysis_runner import run_agent_analysis

result = await run_agent_analysis(
    agent_id=3,                    # analysis_agents 表 ID
    caller="portfolio_panorama",   # 调用标识
    context_parts={                # 上下文片段（自动拼装为 Markdown）
        "持仓明细": holdings_text,
        "估值数据": valuation_text,
    },
    user_question="请诊断我的持仓健康度",
    rag_query="投资组合 资产配置",  # 自动 RAG 检索 + 日志记录
)
# result = {"text": "...", "tokens": 1234, "record_id": 5, "rag_context": "..."}
```

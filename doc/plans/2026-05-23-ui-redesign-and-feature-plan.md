> 状态: Draft

## 一、现状分析

### 当前技术栈

| 层 | 方案 |
|---|---|
| 后端 | FastAPI + akshare + SQLite |
| 前端 | Vue 3 + Vite + Tailwind CSS 4 + ECharts |
| LLM | DeepSeek API |

### 现有页面结构

- **App.vue**: 顶部导航栏 + 主内容区
- **Home.vue**: 两个 Tab（文章管理 / AI 分析）
- **ArticleManagement.vue**: 左侧文章列表 + 右侧详情（图片网格、下载/分析操作）
- **TaskDetail.vue**: 任务详情（文章信息、图片、AI 分析、K 线图、追问）
- **StockChart.vue**: ECharts K 线图
- **ValuationHistory.vue**: 估值历史数据表格 + 区间条

### 当前问题

#### 样式问题
1. **配色不统一**: 部分组件用 `blue` 系，部分用 `indigo` 系，混用严重
2. **间距层次感差**: 卡片之间间距单一（gap-5），缺乏视觉节奏
3. **空状态设计粗糙**: 占位文案居多，缺少插图和引导
4. **Tab 切换样式不一致**: Home.vue 用 pill 样式，TaskDetail.vue 用 underline 样式
5. **表格视觉单调**: 无斑马纹、无 hover 高亮、无排序指示器
6. **K 线图标题样式简陋**: 纯文本 title，缺少图表工具栏
7. **缺少暗色模式**: 只有 light 主题
8. **移动端适配不足**: 文章管理页的左右分栏在小屏下布局异常

#### 功能缺失
1. **无用户认证**: 任何人可访问所有数据
2. **无批量操作**: 文章管理不能批量下载/分析/删除
3. **无导出功能**: 分析结果无法导出 PDF/图片
4. **无搜索高亮**: 搜索结果无关键词高亮
5. **无数据看板**: 缺少投资组合概览、收益统计等 Dashboard
6. **无收藏/标签系统**: 文章无法分类管理
7. **无分析历史对比**: 无法对比不同时期的分析结果
8. **无通知系统**: 后台任务完成后无通知
9. **无 WebSocket**: 任务状态靠轮询，浪费资源
10. **无快捷键支持**: 缺少键盘导航

## 二、UI 重设计方案

### 2.1 设计语言统一

#### 配色方案

```css
/* 主色调 — 深蓝/靛蓝系，符合金融产品调性 */
--primary-50: #eef2ff;
--primary-100: #e0e7ff;
--primary-500: #6366f1;
--primary-600: #4f46e5;
--primary-700: #4338ca;

/* 语义色 */
--success: #10b981;  /* 低估/盈利 */
--warning: #f59e0b;  /* 合理/中性 */
--danger: #ef4444;   /* 高估/亏损 */

/* 中性色 */
--gray-50: #f8fafc;
--gray-100: #f1f5f9;
--gray-200: #e2e8f0;
--gray-500: #64748b;
--gray-900: #0f172a;
```

#### 圆角 & 阴影

```css
/* 卡片 */
--radius-card: 12px;
--shadow-card: 0 1px 3px rgba(0,0,0,0.06), 0 1px 2px rgba(0,0,0,0.04);
--shadow-card-hover: 0 10px 25px rgba(0,0,0,0.08);

/* 按钮 */
--radius-btn: 8px;

/* 输入框 */
--radius-input: 8px;
```

#### 字体层级

```
H1: 24px / bold / gray-900
H2: 18px / semibold / gray-900
H3: 15px / semibold / gray-800
Body: 14px / normal / gray-700
Caption: 12px / normal / gray-500
```

#### 2.2.4 文章管理页改进

- 列表改为卡片流布局，支持列表/网格视图切换
- 批量操作工具栏（全选 → 批量下载/分析/删除）
- 文章标签系统（自定义分类）
- 拖拽排序
- 搜索结果高亮

#### 2.2.5 分析结果页改进

- 分析进度改为步骤条（Step Bar），比 spinner 更有节奏感
- AI 分析结果支持目录导航（ToC）
- K 线图增加 MA 均线叠加、MACD 副图
- 估值区间改为可视化仪表盘（Gauge Chart）
- 追问区改为对话气泡样式，更像 Chat

#### 2.2.6 估值页改进

- 估值区间条改为 Gauge 仪表盘组件
- 历史趋势用折线图替代纯表格
- 增加多指数对比视图
- 增加估值提醒设置

## 三、功能增强方案

### 3.1 P0 — 高优先级

#### 3.1.1 WebSocket 实时推送
- 替代当前轮询方案
- 任务状态变更实时推送到前端
- 降低服务端压力

#### 3.1.2 数据导出
- 分析报告导出为 PDF
- K 线图导出为 PNG
- 估值数据导出为 Excel/CSV
- 一键分享链接

#### 3.1.3 全局搜索 (Cmd+K)
- 搜索文章标题、内容
- 搜索标的代码、名称
- 搜索历史分析记录
- 快捷操作入口

#### 3.1.4 暗色模式
- 跟随系统 / 手动切换
- 图表配色适配
- 减少长时间使用的视觉疲劳

### 3.2 P1 — 中优先级

#### 3.2.1 投资组合管理
- 添加自选标的
- 组合收益计算
- 定时估值更新
- 盈亏分析

#### 3.2.2 分析历史对比
- 选择两次分析结果进行对比
- 标的变化追踪
- 估值趋势变化

#### 3.2.3 通知系统
- 后台任务完成通知（浏览器 Notification API）
- 估值异常提醒（跌破/突破阈值）
- 新文章同步提醒

#### 3.2.4 标签 & 收藏系统
- 文章自定义标签
- 标的收藏夹
- 标签筛选和统计

### 3.3 P2 — 低优先级

#### 3.3.1 用户认证
- 简单密码保护 / JWT 认证
- 多用户支持
- 操作日志

#### 3.3.2 快捷键系统
- J/K 上下导航列表
- Enter 进入详情
- Esc 返回
- Cmd+K 搜索
- Cmd+N 新建分析

#### 3.3.3 移动端优化
- 响应式侧边栏（小屏折叠为 bottom bar）
- 触摸手势支持（左滑删除、下拉刷新）
- PWA 支持（可添加到主屏幕）

#### 3.3.4 分析结果自动刷新
- 定时重新拉取行情数据
- 估值数据自动更新
- 数据变化高亮

## 四、可安装的增强技能/工具

### 4.1 UI 组件库

| 方案 | 说明 | 推荐度 |
|------|------|--------|
| **Naive UI** | Vue 3 原生组件库，暗色模式内置，表格/表单/图表组件丰富 | ⭐⭐⭐⭐⭐ |
| **Element Plus** | 最流行的 Vue 3 UI 库，生态完善 | ⭐⭐⭐⭐ |
| **PrimeVue** | 轻量级，70+ 组件，内置暗色主题 | ⭐⭐⭐⭐ |
| **shadcn-vue** | Vue 版 shadcn/ui，高度可定制，基于 Tailwind | ⭐⭐⭐⭐⭐ |

**推荐**: 如果要保持 Tailwind CSS 生态一致性，选 **shadcn-vue**；如果要开箱即用的丰富组件，选 **Naive UI**。

### 4.2 图表增强

| 方案 | 说明 |
|------|------|
| **ECharts** (已安装) | 可增加 MA/MACD/BOLL 等技术指标叠加 |
| **Lightweight Charts** | TradingView 开源图表库，专业金融图表 |
| **ApexCharts** | 现代图表库，响应式好，动画流畅 |

**推荐**: 继续用 ECharts，增加 `echarts-gl` 做 3D 估值可视化，或引入 `lightweight-charts` 做专业 K 线。

### 4.3 动画 & 交互

| 方案 | 说明 |
|------|------|
| **Motion Vue** | Vue 3 动画库（原 Framer Motion），页面过渡/手势 |
| **VueUse** | 100+ 组合式工具函数（快捷键、暗色模式、本地存储等） |
| **@vueuse/gesture** | 手势识别（拖拽、滑动） |

**推荐**: 安装 **VueUse** — 一站式解决暗色模式、快捷键、本地存储、WebSocket 等大量增强需求。

### 4.4 PDF & 导出

| 方案 | 说明 |
|------|------|
| **html2canvas + jsPDF** | 前端截图 + PDF 生成 |
| **@aspect-ratio/canvas** | 高质量图表截图 |
| **SheetJS (xlsx)** | Excel 导出 |

### 4.5 搜索

| 方案 | 说明 |
|------|------|
| **Fuse.js** | 前端模糊搜索，零后端依赖 |
| **MiniSearch** | 轻量全文搜索 |

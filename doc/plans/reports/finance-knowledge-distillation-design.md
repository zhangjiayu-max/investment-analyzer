
# 理财知识蒸馏流水线设计稿

## 一、蒸馏 Prompt 模板

### 模板 1：从书本提取投资原则

```
你是一个投资知识提炼专家。请从以下投资书籍内容中提取核心原则。

要求：
1. 每条原则一句话概括观点
2. 紧接着用 2-3 句话解释为什么这个原则重要
3. 如果原文有具体数据或例子，保留
4. 每条原则 200-400 字
5. 输出 JSON 数组：[{"principle": "...", "category": "价值投资|资产配置|风险控制|行为金融|..."}]

内容：
{把书的章节贴在这里}
```

### 模板 2：生成 FAQ 问答对

```
你是一个理财教育专家。请基于以下投资知识，生成 20 条新手投资者最常问的问题和回答。

要求：
- 问题用真实用户的口吻（可以口语化、不专业）
- 回答必须有具体数据或数字支撑
- 不能是"建议长期持有"这种废话
- 每条 QA 200-400 字
- 输出 JSON：[{"question": "...", "answer": "...", "tags": ["基金", "定投", ...]}]

参考知识：
{贴你的知识摘要}
```

### 模板 3：从季报提取关键数据

```
请从以下基金季报中提取关键信息：

要求提取：
- 前十大重仓股及占比
- 基金经理对市场的展望（如果有的话）
- 仓位变动（相比上季度增减超过 2% 的股票）
- 行业配置变化

输出结构化 JSON。
季报内容：{贴季报文本}
```

### 模板 4：生成多轮对话训练数据

```
模拟一个理财新手和 AI 理财助手的 5 轮对话。

用户画像：30 岁，月入 2 万，有 10 万存款，刚开始学投资。

第 1 轮：用户问「我应该买什么基金」
第 2 轮：AI 追问风险偏好和投资期限
第 3 轮：用户反问「为什么不能 all in 股票」
第 4 轮：AI 解释分散投资
第 5 轮：用户决定先买沪深 300 + 债券基金

要求每轮对话自然、真实，AI 的回答要有具体数据。
```

## 二、知识库数据规划

### 第一批优先蒸馏（立即见效）

| 类型         |  数量  | 来源                            |
| ------------ | :----: | ------------------------------- |
| FAQ 问答对   | 50 条 | MiMo 自动生成                   |
| 投资术语词典 | 100 条 | PE/PB/ROE/α/β/夏普/最大回撤等 |
| 基金费用规则 | 30 条 | 10 份基金合同提炼               |

### 第二批（深度提升）

| 类型                     |   数量   | 来源     |
| ------------------------ | :------: | -------- |
| 《聪明的投资者》核心原则 | 30-50 条 | 书籍蒸馏 |
| 晨星基金评价方法论       |  20 条  | 公开资料 |
| 估值分位数分析框架       |  20 条  | 行业通用 |

### 第三批（专业壁垒）

| 类型                 | 数量 | 来源         |
| -------------------- | :---: | ------------ |
| 《长赢投资》原则     | 30 条 | 书籍蒸馏     |
| 《漫步华尔街》原则   | 30 条 | 书籍蒸馏     |
| 经济周期与资产轮动   | 20 条 | 美林时钟     |
| Fama-French 因子模型 | 10 条 | 学术文献     |
| 行为金融学偏误对照表 | 20 条 | 行为金融文献 |
| 税收政策             | 20 条 | 财税文件     |

### 不要喂的内容

- 投资建议/荐股（LLM 会当成真实建议输出，合规风险）
- 过时的政策（税法、费率规则可能变化）
- 未经核实的市场分析
- 策略类文章全文（太长，提炼成原则再喂）
- 用户个人财务数据（隐私风险，走结构化数据库）

## 三、摄入流程

```
MiMo 输出的 JSON
    ↓
Python 脚本解析
    ↓
每条 knowledge 片段 → BGE-Large-ZH 向量化
    ↓
存入 Chroma collection "finance_knowledge"
    ↓
Agent 对话时自动检索 Top-5 → 注入上下文
```

## 五、FAQ 示例

### 示例 1

**问**：净值低的基金是不是更便宜？

**答**：基金净值高低和贵不贵没关系。净值 = 总资产 / 总份额。1 元的基金和 5 元的基金，如果你各投 1000 元，涨跌 1% 你赚亏的钱是一样的。买基金看的是基金经理的能力和投资方向，不是看净值。

**标签**：基金, 入门, 误区

### 示例 2

**问**：分红是不是白送钱？

**答**：分红是把你自己的钱从左口袋放到右口袋。基金分红后净值会等额下跌。举例：你持有净值 2 元的基金 1000 份（2000 元），基金分红每份 0.2 元，你拿到 200 元现金，同时净值变成 1.8 元，你的总资产还是 2000 元。分红本身不创造收益，只是把钱从基金里拿出来给你。

**标签**：基金, 分红, 误区

### 示例 3

**问**：定投是不是一定能赚钱？

**答**：定投的本质是分批买入、摊平成本，在市场下跌时买更多份额。但这只解决了"什么时候买"的问题，没解决"买什么"的问题。如果你定投的是一只长期下跌的基金，定投只是在帮你亏更多的钱。定投的前提是你选对了资产——长期向上的宽基指数（如沪深 300、标普 500）才适合定投。

**标签**：定投, 策略, 误区

### 4. 硬性 guardrail（关键决策拦截）

**现状**：当前注入是"软提示"（system message 建议结合画像），`wealth_advisor.py:42-47` 已有 `loss_tolerance` 阈值化雏形（low=5%/medium=15%/high=30%），但仅在主动关怀预警中使用，未在建议生成阶段拦截。

**改造**：在 `pipeline.py` Phase 5 `_extract_and_save_recommendations` 之后，新增 `_apply_kyc_guardrail` 函数：

```python
def _apply_kyc_guardrail(recommendations, user_profile):
    """根据用户画像对建议加 guardrail 标记。

    规则：
    1. risk_tolerance=conservative 且 direction=up 且 confidence=low → 标记 risky_for_profile
    2. loss_tolerance=low 且 direction=down → 不拦截（减仓对低亏损承受者是保护）
    3. risk_tolerance in (conservative, steady) 且 focus_assets 不含 stock 但建议标的为股票指数 → 标记 out_of_focus
    4. max_single_position_pct 已设且建议涉及加仓 → 附加 position_limit_reminder
    """
    for rec in recommendations:
        flags = []
        # 规则1：保守用户 + 低置信度加仓
        if (user_profile.risk_tolerance in ("conservative", "steady")
            and rec.direction == "up" and rec.confidence == "low"):
            flags.append("risky_for_profile")
        # 规则3：超出关注品种
        focus = json.loads(user_profile.focus_assets or "[]")
        if focus and rec.index_code and not _matches_focus(rec.index_code, focus):
            flags.append("out_of_focus")
        # 规则4：持仓上限提醒
        if rec.direction == "up" and user_profile.max_single_position_pct:
            flags.append(f"position_limit:{user_profile.max_single_position_pct}%")
        if flags:
            rec["guardrail_flags"] = flags
    return recommendations
```

**前端**：ChatMessage.vue 建议卡片在 `guardrail_flags` 非空时显示橙色警告条「⚠ 本建议与您的风险偏好不完全匹配（保守型用户+低置信度加仓），请谨慎评估」。

**开关**：`pipeline.kyc_guardrail_enabled`（默认 true）

### 2. 补 volatility 指标

**现状**：`_build_result`（`strategy_library.py:185-211`）只算 total_return / annual_return / max_drawdown / sharpe_ratio，无 volatility。前端期望展示波动率。

**改造**：新增 `_calc_volatility`：
```python
def _calc_volatility(nav_series):
    """年化波动率 = 日收益率标准差 × √252"""
    if len(nav_series) < 2:
        return 0.0
    returns = []
    for i in range(1, len(nav_series)):
        prev = nav_series[i-1]["nav"]
        curr = nav_series[i]["nav"]
        if prev > 0:
            returns.append((curr - prev) / prev)
    if len(returns) < 2:
        return 0.0
    mean_r = sum(returns) / len(returns)
    variance = sum((r - mean_r) ** 2 for r in returns) / (len(returns) - 1)
    return round((variance ** 0.5) * (252 ** 0.5), 4)
```

`_build_result` 返回值新增 `volatility` 字段。

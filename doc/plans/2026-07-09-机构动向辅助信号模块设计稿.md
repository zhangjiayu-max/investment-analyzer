# 机构动向辅助信号模块设计稿

> 日期：2026-07-09
> 目标：**作为辅助信号提升现有建议质量，而非独立跟风策略**
> 原则：滞后效应导致纯跟风回测失真，机构动向的价值在于"与估值/持仓信号共振时增强置信度"

## 一、为什么不做独立跟风策略

| 问题 | 说明 |
|------|------|
| 信息滞后 | 机构持仓披露滞后 15-60 天，等能看到时机构可能已调仓 |
| 择时困难 | 机构建仓是数月过程，一次性买入可能买在建仓中段 |
| 回测失真 | 用未来数据回测会高估收益，实盘无法复制 |

**正确定位**：机构动向 = 辅助确认信号，与估值分位、持仓风险共振时才产生价值。

## 二、数据源选择（4 类，不贪多）

| 数据源 | akshare 接口 | 频率 | 价值 | 接入优先级 |
|--------|-------------|------|------|-----------|
| 北向资金净流入 | `stock_hsgt_north_net_flow_in_em` | 日频 | 实时可查，市场情绪风向标 | P0 |
| 股东户数变动 | `stock_zh_a_gdhs_detail_em` | 季频 | 筹码集中度，户数为下降=机构吸筹 | P1 |
| 龙虎榜机构席位 | `stock_lhb_jgmmtj_em` | 日频 | 机构短期动向，题材股识别 | P1 |
| 大宗交易 | `stock_dzjy_mrmplist` | 日频 | 机构大宗买卖，折价率反映情绪 | P2 |

**不接入的**：
- 十大股东（`stock_gdfx_top_10`）：披露滞后太久，且变动频率低
- 机构持仓报告（`stock_report_disclosure`）：季度披露，实盘价值低

### Layer 3: Agent 层（扩展现有专家，不新建）

**文件**：`backend/db/agents.py`

**决策：不新建 `institutional_analyst` 专家**。原因：
1. 现有 6 个专家已覆盖核心维度，新增专家会增加协调成本
2. 机构动向是"辅助信号"，应注入到现有专家而非独立判断

**方案：扩展 `macro_strategist`（宏观策略师）的 tools 列表**

当前 macro_strategist 的 prompt 提到"北向资金"但无工具（db/agents.py:363）。修复这一断层：
- tools 列表追加 `query_institutional_flow`
- prompt 微调：明确"可调用 query_institutional_flow 获取北向资金/股东户数/龙虎榜/大宗交易数据，作为宏观判断的辅助信号"

**可选扩展**：给 `market_analyst`（投资研究助手）也加该工具，用于短期市场情绪分析。

### Layer 4: guardrail 增强（信号共振检测）

**文件**：`backend/agent/pipeline.py`

在 `_apply_kyc_guardrail` 之后新增 `_apply_institutional_confirm`：

```python
def _apply_institutional_confirm(recommendations: list, trace_id: str) -> list:
    """机构动向共振检测（P0 新增）。
    
    规则：
    - direction=up + 北向资金近5日强净流入 → confidence 提升（不直接改，加 confirm 标记）
    - direction=up + 北向资金近5日强净流出 → 加 guardrail_flags: "against_north_flow"
    - direction=down + 北向资金强净流入 → 加 guardrail_flags: "against_north_flow"
    """
```

**新增 guardrail_flags**：
- `institutional_confirm`：机构动向与建议方向一致（绿色正向标记）
- `against_north_flow`：建议方向与北向资金相反（橙色警示）

**关键设计**：不直接修改 confidence 值（避免污染原始判断），只打标记，前端展示让用户自行决策。

## 七、不做的事（明确边界）

1. **不做独立"跟风策略"回测**——滞后效应导致回测失真
2. **不新建 institutional_analyst 专家**——避免专家膨胀
3. **不直接修改 confidence 值**——只打标记，保留原始判断可追溯
4. **不接入十大股东/机构持仓报告**——披露滞后太久
5. **不做实时推送**——日频数据无需 SSE

## 八、验证标准

1. **数据可用性**：北向资金接口能返回近 30 日序列
2. **工具可调用**：专家分析时能调 `query_institutional_flow` 获取北向数据
3. **guardrail 生效**：建议方向与北向资金相反时出现 `against_north_flow` 标记
4. **前端展示**：TickerBar 显示北向净流入，建议卡片显示共振标记
5. **性能**：北向数据 5 分钟缓存，不拖慢回测/分析

# 理财决策中枢 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first usable slice of the decision center: persist AI/rule suggestions as decisions, expose today's actions through APIs, and show them on the daily dashboard.

**Architecture:** Add a focused backend decision domain (`db/decisions.py` + `routers/decisions.py`) with SQLite CRUD and status transitions. The frontend consumes `/api/decisions/today` through new API helpers and renders a compact "今日行动" dashboard section using existing visual conventions and confirmation patterns. This MVP does not yet implement full LLM extraction, profile evidence, or review automation.

**Tech Stack:** Python 3.11+, FastAPI, sqlite3, Vue 3 Composition API, Vite, Vitest.

---

  return api.get('/decisions/today')

export function updateDecisionStatus(decisionId, status, userNote = '') {
  return api.put(`/decisions/${decisionId}/status`, { status, user_note: userNote })
}

export function completeDecisionAction(decisionId, actionId) {
  return api.post(`/decisions/${decisionId}/actions/${actionId}/complete`)
}

# 债券市场专业知识库

> 用于 AI Agent 分析债券基金和债券市场的结构化知识。

---

## 一、债券品种分类

### 按发行主体

| 类型 | 品种 | 风险特征 |
|------|------|----------|
| **利率债** | 国债、地方政府债、政策性金融债（国开/农发/进出口）、央票 | 信用风险极低，价格主要受利率变动影响 |
| **信用债** | 企业债、公司债、中期票据、短期融资券、定向工具 | 含信用风险，收益率=无风险利率+信用利差 |
| **可转债** | 可转换公司债券 | 含股票期权，兼具股性和债性 |
| **同业存单** | 银行同业存单 | 银行信用，期限1年以内，流动性好 |

### 按期限

| 期限分类 | 久期范围 | 代表品种 |
|----------|----------|----------|
| 短债（货币/短债基金） | 0-1年 | 短期融资券、同业存单、短期限国债 |
| 中短债 | 1-3年 | 中期票据、2-3年期政金债 |
| 中长债 | 3-7年 | 5-7年期国债、信用债 |
| 长债 | 7-10年 | 10年期国债、国开债 |
| 超长债 | 10年以上 | 30年期国债、50年期国债 |

## 二、核心指标

### 收益率指标

| 指标 | 说明 | 分析用途 |
|------|------|----------|
| **到期收益率(YTM)** | 持有至到期的年化回报率 | 评估债券投资价值的基础指标 |
| **票息率** | 债券票面年利率 | 与YTM对比判断折价/溢价 |
| **即期收益率** | 年利息/当前价格 | 快速估算当期收入 |
| **远期收益率** | 未来某时点的预期收益率 | 判断收益率曲线预期 |

### 风险指标

| 指标 | 说明 | 分析用途 |
|------|------|----------|
| **久期(Duration)** | 价格对利率变化的敏感度（1%利率变化引起的价格变化百分比） | 衡量利率风险，久期越长利率风险越大 |
| **修正久期** | 修正后的久期，更精确的价格敏感度 | 债券基金常用久期指标 |
| **凸性(Convexity)** | 久期对利率变化的二阶导数 | 凸性越大的债券在利率下行时受益更多 |
| **信用利差** | 信用债收益率 - 同期限国债收益率 | 衡量信用风险溢价 |
| **信用利差期限结构** | 不同期限的信用利差 | 判断市场对信用风险的定价 |

### 估值指标

| 指标 | 说明 |
|------|------|
| **中债估值** | 中央国债登记结算公司发布的债券估值，是市场标准 |
| **中债指数** | 中债系列债券指数，如中债-综合指数、中债-国债指数 |
| **收益率曲线** | 不同期限的收益率连线，斜率反映利率预期 |

## 三、债券基金分析框架

### 分类识别

```
持仓基金 → 判断是否为债券基金：
  1. 基金名称含"债"、"纯债"、"信用债"、"利率债"、"短债"、"中短债"、"长债"、"可转债"
  2. 基金类型为债券型（fund_type 包含"债券型"）
  3. 持仓中债券占比 > 60%
```

### 债券基金子类型

| 子类型 | 特征 | 风险收益 |
|--------|------|----------|
| **纯债基金** | 100%投资债券，不投股票 | 低风险，年化3-5% |
| - 短债/中短债 | 久期1-3年以内 | 波动极小，年化2-4% |
| - 中长期纯债 | 久期3-10年 | 波动中等，年化3-6% |
| **一级债基** | 可投可转债+少量股票 | 中低风险，年化4-7% |
| **二级债基** | 最多20%股票+80%债券 | 中风险，年化5-8% |
| **可转债基金** | 主要投资可转债 | 中高风险，波动接近权益 |
| **债券指数基金** | 跟踪债券指数（如中债-综合债） | 费率低，跟踪指数 |

### 分析维度

**1. 久期分析**
- 当前债券基金组合的加权久期
- 久期是否匹配当前利率环境：
  - 利率下行期 → 长久期更受益（债券价格上涨更多）
  - 利率上行期 → 短久期更安全（减少价格下跌）
- 判断方法：查看中国10年期国债收益率走势 → 判断利率方向
  - 收益率高位回落 → 利率下行期 → 可考虑中长久期
  - 收益率低位反弹 → 利率上行期 → 建议短久期

**2. 信用风险分析**
- 持仓债券的信用评级分布（AAA/AA+/AA/AA-及以下）
- 信用利差当前水平：
  - 利差高位 → 信用债有配置价值
  - 利差低位 → 信用债性价比不高
- 城投债 vs 产业债的配置比例

**3. 票息收入分析**
- 票息率与同期限国债收益率的差额
- 高票息往往意味着高风险（信用下沉）
- 关注基金票息收入的稳定性

**4. 持有人结构**
- 机构持有比例过高 → 大额赎回风险（流动性风险）
- 个人持有为主 → 稳定性更好

**5. 杠杆率**
- 债券基金可通过回购加杠杆
- 杠杆率=(总资产-现金)/净资产
- 杠杆越高波动越大，收益也越高

## 四、收益率曲线分析

### 曲线形态

| 形态 | 含义 | 投资含义 |
|------|------|----------|
| **正常向上倾斜** | 长期收益率高于短期 | 持有长债获得期限溢价 |
| **平坦化** | 长短端利差收窄 | 市场预期经济放缓 |
| **倒挂** | 短期收益率高于长期 | 通常预示经济衰退 |
| **陡峭化** | 长短端利差扩大 | 经济复苏预期，长债风险上升 |

### 关键期限点

中国国债收益率曲线的关键期限：1Y、2Y、5Y、7Y、10Y、30Y

- **10Y-1Y利差**：最常用的期限利差，衡量经济预期
- **10Y-5Y利差**：中期经济预期
- **30Y-10Y利差**：超长期溢价

### 收益率驱动因素

| 因素 | 对收益率的影响 |
|------|----------------|
| 央行降息/降准 | 收益率下行，债牛 |
| 央行加息/升准 | 收益率上行，债熊 |
| 经济数据向好（PMI/GDP） | 收益率上行 |
| 经济数据走弱 | 收益率下行 |
| 通胀上升（CPI/PPI） | 收益率上行 |
| 通胀下降 | 收益率下行 |
| 股市大涨（股债跷跷板） | 资金分流，收益率上行 |
| 股市大跌 | 避险资金流入，收益率下行 |
| 机构配置需求（月末/季末） | 收益率短期下行 |

## 五、可转债分析

### 核心概念

| 概念 | 说明 |
|------|------|
| **转股溢价率** | 转债价格超出转股价值的比例，溢价率越低股性越强 |
| **纯债溢价率** | 转债价格超出纯债价值的比例，衡量债底保护 |
| **纯债价值** | 将转债当作纯债估值的价格，即债底 |
| **Delta** | 转债价格对正股价格的敏感度 |
| **强制赎回(强赎)** | 正股连续30天中有15天超过转股价130%时，发行方可强制赎回 |
| **下修** | 公司主动下调转股价，促进转股 |

### 常见策略

- **双低策略**：选择价格低（<120元）+转股溢价率低（<20%）的转债
- **债底保护策略**：选择价格接近纯债价值的转债，下行有限
- **进攻型策略**：选择低溢价率高 Delta 的转债，跟随正股上涨
- **打新策略**：可转债打新，中签上市首日卖出

## 六、债券市场温度与周期

### 债市温度计解读

有知有行债市温度计：
- **0-30度**：债市偏冷，债券有投资价值，可考虑增加债券配置
- **30-70度**：债市适中，正常配置即可
- **70-100度**：债市偏热，债券价格偏高，需谨慎

注意：债市温度与股市温度的解读不同。债市温度低=债券收益率高=债券价格低=买入机会。

### 债券市场周期

| 阶段 | 特征 | 应对 |
|------|------|------|
| 牛市初期 | 经济放缓，央行开始宽松，收益率高位回落 | 拉长久期，配置长债 |
| 牛市中期 | 收益率持续下行，债券价格上涨 | 保持久期，获取资本利得 |
| 牛市末期 | 收益率极低，债市温度高 | 缩短久期，减配长债 |
| 熊市初期 | 经济复苏，央行收紧，收益率低位反弹 | 缩短久期，增配短债/货基 |
| 熊市中期 | 收益率快速上行，债券价格下跌 | 短久期防御，等待机会 |
| 熊市末期 | 收益率高位，债市温度低 | 逐步拉长久期，布局 |

## 七、持仓组合中的债券配置

### 配置原则

- **股债跷跷板**：股票和债券通常负相关，组合配置降低波动
- **风险平价**：根据波动率分配股债比例，使两类资产贡献的风险相等
- **目标日期策略**：随投资期限临近逐步增加债券比例
- **杠铃策略**：同时配置超短债和超长债，中间期限少配

### 债券基金组合构建

```
零钱/活钱（0-3月）→ 货币基金 / 同业存单基金 / 短债基金
短期资金（3-12月）→ 中短债基金 / 短期信用债
中期配置（1-3年）→ 中长期纯债基金 / 二级债基
长期配置（3年以上）→ 长债基金 / 可转债基金 / 债券指数基金
```

### 风险监测

每月检查：
1. 组合加权久期是否匹配利率方向判断
2. 信用债占比与信用利差水平是否匹配
3. 可转债仓位是否超出风险承受能力
4. 杠杆率是否过高（单只基金杠杆>120%需警惕）
5. 单只基金集中度（单只基金占比超过30%需分散）

## 八、当前市场要点

### 数据源说明

- **债市温度**：有知有行债券市场温度计（每日更新），反映债市整体估值水平
- **国债收益率**：akshare bond_zh_us_rate 获取中国国债收益率曲线数据
- **银行间利率**：Shibor 利率反映银行间资金面松紧
- **盈米 MCP**：通过盈米基金诊断接口获取基金持仓中债券明细
- **基金持仓**：akshare fund_portfolio_bond_hold_em 获取基金债券持仓

### 分析切入点

当用户询问债券相关问题，需要：
1. 先判断是问具体债券基金，还是整体债市，还是组合配置
2. 具体基金 → 查基金类型+持仓债券明细+久期信息
3. 整体债市 → 查债市温度+收益率曲线+资金面
4. 组合配置 → 查当前股债比例+久期匹配度+信用风险暴露
